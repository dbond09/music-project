# Riffster

Web-based app to share simple melodies.

[![](http://img.youtube.com/vi/hHMTk73yIsM/0.jpg)](http://www.youtube.com/watch?v=hHMTk73yIsM "")

## Getting Started
```sh
npm install
npm start
```

URL: http://localhost:3000/
default credentials:
  doctorwhocomposer
  password
Works best in Google Chrome

Run tests:
```sh
npm test
```
